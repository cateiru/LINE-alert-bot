## TL;DR

[気象庁から地震のデータ](http://www.data.jma.go.jp/developer/xml/feed/eqvol.xml)を取得してLINEにpostします。

## 💻 Environment

- Python: 3.6

## ⚙ Installing dependencies

Install from Pipenv.

```bash
pip install pipenv

pipenv install
```

## 🚀 Run

```bash
pipenv shell
python src/main.py
```
